# Ninja Island

## Overview
Ninja Island is a fully playable 2D action game demo inspired by classic
NES-era action-adventure games.
The project focuses on custom game logic, AI behavior and state management,
implemented entirely through code.

## Tech Stack
- Java
- JavaFX / Swing (se usati)
- Object-Oriented Programming
- Custom AI logic

## Gameplay Features
- 3 playable levels
- 5 enemy types with different behaviors
- 1 boss fight
- Dynamic map system
- Shop/upgrade system
- Audio system (SFX + game music)
- Main menu and in-game HUD

## AI and Game Logic
- Enemy behavior implemented manually via code
- Pattern recognition and state-based AI
- No reliance on physics colliders for interactions
- Event-driven interactions

## Software Engineering Focus
- OOP and modular design
- State management
- Input handling
- Separation between game logic and presentation
- Debugging and iterative development

## Play the Demo
The game is fully playable locally.
